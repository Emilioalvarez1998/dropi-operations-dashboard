# 📊 Dropi Operations Dashboard (Power BI)

## 📌 Descripción
Dashboard operativo desarrollado en Power BI para el seguimiento diario de envíos, devoluciones y desempeño de dropshippers y proveedores.

## 🎯 Objetivo
Monitorear la operación logística y detectar problemas en:
- Entregas
- Devoluciones
- Estado de pedidos
- Rendimiento por actor (dropshipper/proveedor)

## 🧠 Herramientas utilizadas
- Power BI
- DAX
- Modelado de datos

## 📊 Análisis incluidos
- Envíos por dropshipper
- Envíos por proveedor
- Envíos diarios por región
- Estado de pedidos (entregado, devolución, en proceso)
- Productos más enviados
- Dropshippers sin entregas
